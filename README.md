# E-Store - Full-Stack E-Commerce Application (In Progress)

A backend e-commerce platform built with Node.js and Express.js following the MVC architecture pattern. The project demonstrates progressive database integration, starting with MySQL/Sequelize and evolving to MongoDB,and will include advance Node approaches as authentication,sessions,and validation.

## Features

- **Product Management** - Full CRUD operations (Create, Read, Update, Delete) for products
- **Shopping Cart** - Add products to cart with quantity tracking
- **Order Management** - Place orders and view order history
- **Admin Panel** - Dedicated admin interface for managing products
- **Dynamic Routing** - RESTful route structure with parameterized URLs
- **Server-Side Rendering** - EJS templating engine for dynamic HTML views

## Tech Stack

- **Runtime:** Node.js
- **Framework:** Express.js
- **Templating:** EJS
- **Databases:** MongoDB, MySQL
- **ODM/ORM:** Mongoose, Sequelize
- **Architecture:** MVC (Model-View-Controller)
